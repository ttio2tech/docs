## Get started with Nunchaku Image API

Follow this guide to make your first API request and generate an AI image.

### Step 1: Get your API key

<Card icon="key" title="Obtain your API key">
  Contact your administrator or sign up to receive your API key. You'll use this key to authenticate all API requests.
</Card>

<Note>
  Keep your API key secure and never share it publicly. Treat it like a password.
</Note>

### Step 2: Make your first request

Use this curl command to generate your first image:

```bash
curl -X POST http://74.81.65.159:30371/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "model": "nunchaku-flux.2-klein-9b",
    "tier": "normal",
    "prompt": "A beautiful sunset over mountains",
    "n": 1,
    "size": "1024x1024",
    "response_format": "b64_json",
    "seed": 42
  }'
```

<Warning>
  Replace `YOUR_API_KEY` with your actual API key before running the command.
</Warning>

### Step 3: Decode and save the image

The API returns a JSON response with the generated image. When using `response_format: "b64_json"`, the response looks like:

```json
{
  "data": [
    {
      "b64_json": "iVBORw0KGgoAAAANSUhEUgAA..."
    }
  ],
  "created": 1234567890,
  "model": "nunchaku-flux.2-klein-9b"
}
```

To decode and save the image:

```bash
# Using Python
python -c "
import json, base64, sys
data = json.load(sys.stdin)
open('output.png', 'wb').write(base64.b64decode(data['data'][0]['b64_json']))
" < response.json

# Or using jq and base64
cat response.json | jq -r '.data[0].b64_json' | base64 -d > output.png
```

## Using Python and OpenAI SDK

For Python projects, you can use the OpenAI SDK for easier integration:

<CodeGroup>

```python Requests Library
import base64
from pathlib import Path

import requests

url = "http://74.81.65.159:30371/v1/images/generations"
headers = {
    "X-API-Key": "YOUR_API_KEY",
    "Content-Type": "application/json",
}

payload = {
    "model": "nunchaku-flux.2-klein-9b",
    "tier": "normal",
    "prompt": "A beautiful sunset over mountains",
    "n": 1,
    "size": "1024x1024",
    "response_format": "b64_json",
    "seed": 42,
}

response = requests.post(url, json=payload, headers=headers)
response.raise_for_status()

result = response.json()
image_b64 = result["data"][0]["b64_json"]
Path("output.png").write_bytes(base64.b64decode(image_b64))
```

```python OpenAI SDK
import base64
from pathlib import Path

from openai import OpenAI

# pip install openai
client = OpenAI(
    base_url="http://74.81.65.159:30371/v1",
    api_key="YOUR_API_KEY",
)

response = client.images.generate(
    model="nunchaku-flux.2-klein-9b",
    prompt="A beautiful sunset over mountains",
    size="1024x1024",
    response_format="b64_json",
    n=1,
    extra_body={
        "tier": "normal",
        "seed": 42,
    },
)

image_b64 = response.data[0].b64_json
Path("output.png").write_bytes(base64.b64decode(image_b64))
```

</CodeGroup>

## Alternative: URL response format

You can also receive the image as a URL instead of base64:

```bash
curl -X POST http://74.81.65.159:30371/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "model": "nunchaku-flux.2-klein-9b",
    "tier": "normal",
    "prompt": "A beautiful sunset over mountains",
    "response_format": "url"
  }'
```

Response:
```json
{
  "data": [
    {
      "url": "https://example.com/generated-image.png"
    }
  ]
}
```

## Common use cases

<AccordionGroup>
  <Accordion title="Generate product mockups" icon="box">
    Use the API to create product visualization images for e-commerce, marketing materials, or presentations.
    
    ```bash
    curl -X POST http://74.81.65.159:30371/v1/images/generations \
      -H "X-API-Key: YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "nunchaku-flux.2-klein-9b",
        "tier": "faster",
        "prompt": "professional product photo of a sleek wireless headphone on white background",
        "size": "1024x1024"
      }'
    ```
  </Accordion>
  
  <Accordion title="Edit existing images" icon="wand-magic-sparkles">
    Transform photos by adding, removing, or modifying elements using natural language.
    
    ```bash
    # First, base64-encode your input image:
    # base64 -w 0 input.png > input_b64.txt

    curl -X POST http://74.81.65.159:30371/v1/images/edits \
      -H "Content-Type: application/json" \
      -H "X-API-Key: YOUR_API_KEY" \
      -d '{
        "model": "nunchaku-flux.2-klein-9b-edit",
        "tier": "normal",
        "prompt": "add a sun in the sky",
        "url": "data:image/png;base64,<BASE64_IMAGE_DATA>",
        "n": 1,
        "size": "1024x1024",
        "response_format": "b64_json",
        "seed": 42
      }'
    ```
  </Accordion>
  
  <Accordion title="Batch generation" icon="grid">
    Generate multiple variations by changing the seed parameter or adjusting prompts.
    
    ```bash
    # Generate 3 variations with different seeds
    for seed in 42 123 456; do
      curl -X POST http://74.81.65.159:30371/v1/images/generations \
        -H "Content-Type: application/json" \
        -H "X-API-Key: YOUR_API_KEY" \
        -d "{
          \"model\": \"nunchaku-flux.2-klein-9b\",
          \"tier\": \"normal\",
          \"prompt\": \"abstract art with vibrant colors\",
          \"seed\": $seed,
          \"response_format\": \"b64_json\"
        }" | jq -r '.data[0].b64_json' | base64 -d > "output_${seed}.png"
    done
    ```
  </Accordion>
