## Available models

The Nunchaku Image API offers several state-of-the-art AI models for image generation and editing. Each model has unique characteristics optimized for different use cases.

## Text-to-image models

### Qwen Image

High-quality image generation with excellent prompt understanding and detail rendering.

<Card title="nunchaku-qwen-image" icon="sparkles">
  **Best for**: High-quality generation, complex prompts, detailed scenes
  
  **Supported tiers**: radically_fast, faster, fast, normal
  
  **Strengths**:
  - Superior prompt comprehension
  - Rich detail and texture
  - Consistent output quality
  - Supports all speed tiers
</Card>

```bash
curl -X POST http://74.81.65.159:30371/v1/images/generations \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nunchaku-qwen-image",
    "tier": "fast",
    "prompt": "a red apple on a wooden table",
    "size": "1024x1024"
  }'
```

## Image-to-image (editing) models

### Qwen Image Edit

Edit and transform existing images with natural language instructions.

<CardGroup cols={2}>
  <Card title="nunchaku-qwen-image-edit" icon="paintbrush">
    Standard editing model with excellent instruction following
    
    **Supported tiers**: All tiers
  </Card>
  
  <Card title="nunchaku-qwen-image-edit-2509" icon="wand-magic-sparkles">
    Enhanced 2509 variant with improved capabilities
    
    **Supported tiers**: All tiers
  </Card>
</CardGroup>

```bash
# Encode your input image
IMG_B64=$(base64 -w 0 input.png)

# Send edit request
curl -X POST http://74.81.65.159:30371/v1/images/edits \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d "{
    \"model\": \"nunchaku-qwen-image-edit\",
    \"tier\": \"fast\",
    \"prompt\": \"add a sun in the sky\",
    \"url\": \"data:image/png;base64,${IMG_B64}\",
    \"size\": \"1024x1024\"
  }"
```

## Model variants

Some models offer multiple variants optimized for different scenarios:

### Base variants
- Standard models with default capabilities
- Examples: `nunchaku-flux.2-klein-9b-base`, `nunchaku-flux.2-klein-4b-base`

### Edit variants
- Specialized for image-to-image transformation
- Examples: `nunchaku-qwen-image-edit-2509`, `nunchaku-flux.2-klein-9b-edit`

<Tip>
  Start with `nunchaku-qwen-image` with the `fast` tier for a good balance of quality and speed, then adjust based on your specific needs.
</Tip>
