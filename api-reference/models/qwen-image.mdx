## Performance characteristics

| Tier | Generation Time | Quality | Best Use Case |
|------|----------------|---------|---------------|
| `radically_fast` | ~1-2s | Good | Quick previews, iterations |
| `faster` | ~2-3s | Better | Rapid prototyping |
| `fast` | ~3-5s | High | Production use (recommended) |
| `normal` | ~6-10s | Highest | Final outputs, hero images |

<Note>
  Times are approximate and may vary based on prompt complexity and system load.
</Note>

## Example usage

### Text-to-image generation

```bash
curl -s -X POST http://74.81.65.159:30371/v1/images/generations \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nunchaku-qwen-image",
    "tier": "fast",
    "prompt": "professional product photo of sleek wireless headphones, white background, studio lighting, commercial quality",
    "size": "1024x1024",
    "response_format": "b64_json",
    "seed": 42
  }' | jq -r '.data[0].b64_json' | base64 -d > qwen-headphones.png
```

### Image editing

```bash
IMG_B64=$(base64 -w 0 input.png)

curl -s -X POST http://74.81.65.159:30371/v1/images/edits \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d "{
    \"model\": \"nunchaku-qwen-image-edit\",
    \"tier\": \"fast\",
    \"prompt\": \"add a bright sun in the sky and enhance the colors to be more vibrant\",
    \"url\": \"data:image/png;base64,${IMG_B64}\",
    \"size\": \"1024x1024\",
    \"response_format\": \"b64_json\"
  }" | jq -r '.data[0].b64_json' | base64 -d > qwen-edited.png
```

## Tips for best results

<Tip>
  **Prompt engineering**: Qwen models understand detailed, descriptive prompts. Include specifics about style, lighting, composition, colors, materials, and atmosphere for best results.
</Tip>

<Tip>
  **Tier selection**: Start with `tier: "fast"` for a good balance. Use `normal` when maximum quality is essential for final outputs.
</Tip>

<Tip>
  **Image editing**: Be specific about what to change, add, or remove. Qwen models can handle complex multi-part instructions.
</Tip>
