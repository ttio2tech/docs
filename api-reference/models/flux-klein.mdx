## Overview

FLUX Klein models offer excellent performance with faster generation times compared to Qwen models. Available in 4B and 9B parameter variants, these models provide a great balance between quality and speed.

## Performance characteristics

### 9B Models

| Tier | Generation Time | Quality | Best Use Case |
|------|----------------|---------|---------------|
| `faster` | ~1-2s | Better | Quick iterations |
| `fast` | ~2-3s | High | Production use (recommended) |
| `normal` | ~3-5s | Very High | Final outputs |

### 4B Models

| Tier | Generation Time | Quality | Best Use Case |
|------|----------------|---------|---------------|
| `faster` | ~0.5-1s | Good | Real-time apps, rapid generation |
| `fast` | ~1-2s | Better | Quick production work |
| `normal` | ~2-3s | High | Quality outputs with speed |

<Note>
  FLUX Klein models do not support the `radically_fast` tier.
</Note>

## Example usage

### FLUX Klein 9B generation

```bash
curl -s -X POST http://74.81.65.159:30371/v1/images/generations \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nunchaku-flux.2-klein-9b",
    "tier": "fast",
    "prompt": "cozy coffee shop interior with vintage furniture",
    "size": "1024x1024",
    "response_format": "b64_json",
    "seed": 42
  }' | jq -r '.data[0].b64_json' | base64 -d > flux9b-coffeeshop.png
```

### FLUX Klein 4B generation

```bash
curl -s -X POST http://74.81.65.159:30371/v1/images/generations \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nunchaku-flux.2-klein-4b",
    "tier": "faster",
    "prompt": "simple minimalist icon design",
    "size": "1024x1024",
    "response_format": "b64_json",
    "seed": 42
  }' | jq -r '.data[0].b64_json' | base64 -d > flux4b-icon.png
```

### FLUX Klein editing

```bash
IMG_B64=$(base64 -w 0 input.png)

curl -s -X POST http://74.81.65.159:30371/v1/images/edits \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d "{
    \"model\": \"nunchaku-flux.2-klein-9b-edit\",
    \"tier\": \"fast\",
    \"prompt\": \"add colorful flowers in the foreground\",
    \"url\": \"data:image/png;base64,${IMG_B64}\",
    \"response_format\": \"b64_json\"
  }" | jq -r '.data[0].b64_json' | base64 -d > flux9b-flowers.png
```

## Model comparison

### 9B vs 4B

| Feature | 9B Models | 4B Models |
|---------|-----------|-----------|
| **Quality** | High | Good |
| **Speed** | Fast | Very Fast |
| **Detail** | Better | Good |
| **Complex prompts** | Better | Good |
| **Best for** | Balanced work | Speed priority |

### FLUX vs Qwen

| Feature | FLUX Klein | Qwen |
|---------|------------|------|
| **Speed** | Faster | Slower |
| **Quality** | High | Highest |
| **Detail** | Good | Excellent |
| **Tiers** | 3 tiers | 4 tiers (includes radically_fast) |

## Best practices

<Tip>
  **Choose 9B for production**: The 9B variant offers the best balance of quality and speed for most applications.
</Tip>

<Tip>
  **Use 4B for speed**: When speed is critical and prompts are relatively simple, the 4B variant delivers excellent performance.
</Tip>

<Tip>
  **Tier selection**: Start with `tier: "fast"` for the 9B model and `tier: "faster"` for the 4B model.
</Tip>
