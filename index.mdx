## Available models

<AccordionGroup>
  <Accordion title="Qwen Image" icon="sparkles">
    High-quality image generation and editing with support for all speed tiers. Includes both base and 2509 variants.
    - **Text-to-image**: `nunchaku-qwen-image`
    - **Image editing**: `nunchaku-qwen-image-edit`, `nunchaku-qwen-image-edit-2509`
  </Accordion>
  
  <Accordion title="FLUX Klein" icon="bolt">
    Fast and efficient generation with 4B and 9B parameter variants.
    - **Text-to-image**: `nunchaku-flux.2-klein-4b`, `nunchaku-flux.2-klein-9b`
    - **Image editing**: `nunchaku-flux.2-klein-4b-edit`, `nunchaku-flux.2-klein-9b-edit`
  </Accordion>
  
  <Accordion title="Z-Image Turbo" icon="rocket">
    Ultra-fast generation optimized for speed.
    - **Text-to-image**: `nunchaku-z-image-turbo`
  </Accordion>
</AccordionGroup>

## Quick example

Generate your first image with a simple curl command:

```bash
curl -X POST http://74.81.65.159:30371/v1/images/generations \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nunchaku-qwen-image",
    "tier": "fast",
    "prompt": "a serene mountain landscape at sunset",
    "size": "1024x1024",
    "response_format": "b64_json"
  }'
```
